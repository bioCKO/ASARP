NAME

asarp.pl -- The main application script, i.e. the entry program, of the ASARP pipeline.

SYNOPSIS

Look at the source: ../asarp.pl and it is self-explanatory. There are basically 3 steps:

1. parse the input files and compile alternative mRNA processing events. see fileParser

2. get the SNVs and match them with the events. see snpParser

3. process ASARP (including ASE) patterns and output the formatted results. see source and snpParser

DESCRIPTION

The methodology in detail is explained in the paper:

Li G, Bahn JH, Lee JH, Peng G, Chen Z, Nelson SF, Xiao X. Identification of allele-specific alternative mRNA processing via transcriptome sequencing, Nucleic Acids Research, 2012, 40(13), e104

and its Supplementary Materials

See nar.oxfordjournals.org for more details of the paper.
